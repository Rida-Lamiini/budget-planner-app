# Budget Planner App

The Budget Planner App is a mobile application designed to help users manage their finances effectively. It allows users to categorize their expenses, track spending patterns, and make informed financial decisions on the go.

![App Screenshot](./screenshots/app-screenshot.png)

## Features

- **Category Management**: Create and manage expense categories such as groceries, utilities, entertainment, etc.
  
- **Expense Tracking**: Record and track expenses within each category to monitor spending habits.
  
- **Visual Analytics**: Visual representation of expenditure through charts and graphs for better understanding.
  
- **URL Management**: Link budget items to external resources for easy access to additional information.

## Technologies Used

- **React Native**: Frontend framework for building cross-platform mobile applications.
  
- **Supabase**: Backend as a service (BaaS) for managing data storage and authentication.
  
- **Expo**: Development toolchain and platform for simplifying React Native development.

## Installation

To run the Budget Planner App locally on your machine, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Rida-Lamiini/budget-planner-app.git
   cd budget-planner-app
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**

   Create a `.env` file in the root directory and configure your Supabase credentials:

   ```env
   REACT_NATIVE_SUPABASE_URL=https://your-supabase-url.supabase.co
   REACT_NATIVE_SUPABASE_KEY=your-supabase-api-key
   ```

4. **Run the application:**

   ```bash
   npm start
   ```

   Follow the instructions from Expo CLI to launch the app on an emulator or physical device.

## Usage

- **Creating Categories**: Add new expense categories through the app interface.
  
- **Adding Expenses**: Within each category, add new expenses with details like cost, description, and URL links.
  
- **Viewing Analytics**: Monitor spending patterns and visualize expenditure using the built-in charts.
